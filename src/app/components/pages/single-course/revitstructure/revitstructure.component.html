Autodesk's Revit Structure is a BIM software solution designed for structural engineering companies and engineers. Its feature-rich toolset enables efficient design processes in a BIM environment and facilitates collaboration with other construction disciplines using Autodesk software.

Course Description

Revit Building Information Modeling software can streamline projects from design concept to fabrication, improving installation accuracy and constructability by connecting structural design to the detailed model.

Revit Structure is a powerful platform for structural engineering, providing cutting-edge tools for creating design alternatives, building simulations, and publishing. Our training course is based on a Revit structural template, allowing you to learn how to annotate, detail your model, create and print sheets, import CAD files, and much more.

The Revit structure training course includes BIM software specifically for structural engineering, which enhances structural design coordination, minimizes errors, increases overall efficiency and profitability, and produces high-quality outcomes.

The objective of this training course is to provide an introduction to the user interface and basic building components of Autodesk Revit, which is a powerful and flexible structural modeling tool. The course aims to familiarize you with the tools needed to create, modify, analyze, and document a parametric model.

The training course covers various topics, including navigating the user interface, linking architecture, creating levels, creating view templates, adding and placing steel columns, creating foundations with footings, piers, pilasters, and slabs, designing retaining walls, adding beam systems, tagging steel framing, creating a slab floor, cantilevering slab edges, reinforcing with rebar, adding brace frames, creating stairs and ramps, detailing and annotating drawings, inserting AutoCAD geometry, creating schedules, adding and attaching trusses, and plotting and sharing files.

By the end of the course, you will have gained knowledge on how to effectively use Autodesk Revit to create, modify, and document a structural model with parametric design capabilities.
